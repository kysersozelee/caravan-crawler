# caravan-crawler
[네이버 데이터랩](https://datalab.naver.com/shoppingInsight/sCategory.naver) 크롤링 프로젝트

## development environment
- pyenv
- python 3.7.0

## install
```bash
# activate venv
$ python3 -m venv ./venv
$ source ./venv/bin/activate

# install requirements
$ pip3 install -r requirements.txt

# deactivate
$ deactivate
```
